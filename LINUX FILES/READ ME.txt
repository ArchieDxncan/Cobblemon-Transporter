Cobblemon-Transporter now supports Linux natively without Wine!

The application uses .NET DLLs that run directly on Linux with the .NET runtime installed.

Prerequisites:
- Python 3.8+
- .NET 9.0 runtime
- tkinter support

Installation:
1. Copy these files into the main Cobblemon Transporter folder
2. Run: chmod +x install.sh start.sh
3. Run: ./install.sh
4. Run: ./start.sh

For .NET installation on Ubuntu/Debian:
wget https://packages.microsoft.com/config/ubuntu/20.04/packages-microsoft-prod.deb -O packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
rm packages-microsoft-prod.deb
sudo apt-get update
sudo apt-get install -y dotnet-runtime-9.0

The old Wine method is no longer needed, but if you prefer Wine, ensure ~/.winepkhex exists with dotnetdesktop9:
env WINEPREFIX=/home/$USER/.winepkhex winetricks dotnetdesktop9
